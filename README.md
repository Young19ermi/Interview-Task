## **ğŸ¨ How to Use**

1. **Add Shape**: Use the dropdown to add a Sphere, Cube, or Cylinder.
2. **Select**: Click on any object to select it.
3. **Transform**:

   * **Move**: Drag the object with the mouse.
   * **Rotate**: Use the rotation sliders to spin the object.
   * **Scale**: Use the scaling sliders to resize it.
4. **Scene Controls**:

   * **Save**: Save your current scene as `.json`
   * **Load**: Load a previously saved scene
   * **Import OBJ**: Load external 3D models (.obj)
   * **Export OBJ**: Export the selected object as `.obj`
