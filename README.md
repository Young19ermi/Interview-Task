# **🟢 3D Object Editor**

An interactive web-based 3D editor built with **Three.js** that allows users to add, move, rotate, and scale objects in a 3D scene using intuitive mouse and UI controls.

---

## **✨ Features**

* 🟢 **Add Shapes** – Sphere, Cube, Cylinder
* 🎯 **Click to Select** – Interact with individual objects
* 🖱️ **Move with Mouse Drag** – Easily reposition objects by dragging
* 🔄 **Rotate with Sliders** – Smooth and precise object rotation
* 📏 **Scale with Sliders** – Uniform and axis-based scaling
* 💾 **Save & Load Scene** – Export and import your layout as `.json`
* 📥 **OBJ Import & Export** – Work with standard 3D object files
* 🧹 **Undo (Remove Last)** – Quick undo for recent shape additions
* 🎛️ **Live Object Info** – View position, rotation, and scale values in real-time

---

## **🛠️ Technologies Used**

* [Three.js](https://threejs.org/) – 3D rendering engine
* **Vanilla JavaScript**
* **HTML/CSS**

## **🎨 How to Use**

1. **Add Shape**: Use the dropdown to add a Sphere, Cube, or Cylinder.
2. **Select**: Click on any object to select it.
3. **Transform**:

   * **Move**: Drag the object with the mouse.
   * **Rotate**: Use the rotation sliders to spin the object.
   * **Scale**: Use the scaling sliders to resize it.
4. **Scene Controls**:

   * **Save**: Save your current scene as `.json`
   * **Load**: Load a previously saved scene
   * **Import OBJ**: Load external 3D models (.obj)
   * **Export OBJ**: Export the selected object as `.obj`
